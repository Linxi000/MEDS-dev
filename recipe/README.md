# Recipe: MEDS

This directory contains the implementation of **MEDS** (Memory-Enhanced Diversity Sampling), our RL training recipe based on the [verl](https://github.com/volcengine/verl) framework.

## Structure

```
recipe/
└── meds/         
    ├── config/      
    ├── main_meds.py   
    ├── meds_ray_trainer.py     
    ├── extract_layer_logits.py 
    ├── layer_logits_utils.py   
    └── run_meds.sh    
```
